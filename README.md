# Web-deb-2 final project

## Struture
- Backend
    - express v4.18.2
    - nodemailer
    - jwt
    - mongoose
- Frontend
    - vite v5.1.4
    - react v18.2.0
    - tailwindcss v3.4.1

## Installation
- Clone repository
    ```bash
    git clone https://github.com/helloWRLDs/web-final.git
    ```


- Execute in the root folder
    ```bash
    npm run install
    ```
